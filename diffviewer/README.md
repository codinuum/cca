# DiffViewer for Diff/AST

## Requirements

* [Node.js](https://nodejs.org/en/download/)

## Build (macOS)

    $ npm install
    $ ./mkpkg.sh

## Usage (macOS)

    $ ../src/ast/analyzing/bin/diffast [--cache PATH] PATH0 PATH1
    $ ./run.py [--cache PATH] PATH0 PATH1
